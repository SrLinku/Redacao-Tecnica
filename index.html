Checklist de Boas Práticas - Dados Sensíveis

  • Informar o usuário sobre como os dados serão usados
  • Utilizar HTTPS sempre que houver envio de dados
  • Evitar armazenar senhas em campos visíveis
  • Nunca salvar dados sensíveis no LocalStorage
  • Sanitizar e validar entradas do usuário
  • Utilizar autenticação de múltiplos fatores quando possível
  • Evitar exibir dados sensíveis no frontend
Este site coleta dados pessoais. Ao prosseguir, você concorda com nossa política de privacidade.
